# Quick Start: Requester

Sugdi ang paghimo og disbursement vouchers sulod sa 5 minutos!

## Unsa ang Imong Mahimo

Himoa ug i-submit ang disbursement voucher para sa review.

## Prerequisites

- ✅ Nakalog-in ka na sa sistema
- ✅ Adunay ka REQUESTER role
- ✅ Nahibalo ka sa payee ug payment details

## Essential Steps

### Step 1: Ablihi ang Create Request Page

I-click ang **"Create Request"** sa navigation menu.

### Step 2: Pun-i ang Form

1. **Select Payee**: I-click ang Payee field, type ang ngalan, pilia gikan sa dropdown
2. **Enter Address**: Type ang kompleto nga payee address
3. **Enter Particulars**: I-describe kung para sa unsa ang payment (pananglitan, "Office supplies - January 2024")
4. **Enter Amount**: Type ang amount sa pesos (pananglitan, `5000` para sa ₱5,000.00)
5. **Add Remarks** (optional): Bisan unsa nga additional notes

### Step 3: Submit para sa Review

I-click ang **"Submit for Review"** button.

**Tapos na!** Ang imong voucher karon PENDING na ug ang Secretary madunggan na.

## Unsa ang Imong Gilauman

- ✅ Ang voucher status mausab ngadto sa PENDING
- ✅ Ang Secretary makadawat og notification
- ✅ Makadawat ka og notifications samtang nag-progress ang voucher
- ✅ Track ang progress sa "My Vouchers"

## Common First-Time Mistakes

❌ **Pag-submit og incomplete information**: Pun-i ang tanan nga required fields
❌ **Sayop nga amount**: I-double-check una sa pag-submit
❌ **Vague particulars**: Specific sa payment purpose
