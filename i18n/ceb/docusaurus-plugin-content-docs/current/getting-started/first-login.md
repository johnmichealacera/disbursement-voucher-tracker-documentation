# First-Time Login

Maayong pag-abot! Kining giya makatabang kanimo nga ma-access ang Disbursement Voucher Tracker sa unang higayon ug makasabot sa interface.

## Pag-Access sa Application

### Step 1: Ablihi ang Imong Web Browser

Ablihi ang imong preferred web browser. Girekomenda namo ang paggamit sa **Google Chrome** o **Microsoft Edge** para sa pinakamaayong experience.

### Step 2: Navigate sa Application URL

Ang imong system administrator maghatag kanimo sa web address (URL) para sa Disbursement Voucher Tracker. Kini kasagaran magtan-aw sama sa:
- `https://voucher-tracker.yourmunicipality.gov.ph`
- O susama nga web address

I-type kini nga address sa imong browser's address bar ug i-press ang **Enter**.

### Step 3: Ang Login Page

Makita nimo ang login page uban sa:
- **Username** field
- **Password** field
- **Login** button

:::info Security Note
Ang login page naggamit og secure encryption. Tan-awa ang padlock icon (ðŸ”’) sa imong browser's address bar aron ma-confirm nga secure ang connection.
:::

### Step 4: I-enter ang Imong Credentials

1. I-click ang **Username** field
2. I-type ang imong username (gihimo sa imong administrator)
3. I-click ang **Password** field
4. I-type ang imong password (gihimo sa imong administrator)

:::warning Password Security
- Ang passwords case-sensitive (capital ug lowercase letters importante)
- Siguroha nga ang Caps Lock wala enabled
- Kung dili nimo makita ang imong gina-type, normal naâ€”ang passwords gitago para sa security
:::

### Step 5: I-click ang Login

I-click ang **Login** button aron ma-access ang sistema.

### Step 6: First Login Success

Kung tama ang imong credentials, ma-redirect ka sa imong **Dashboard**. Congratulationsâ€”naa ka na sa sulod!

:::danger Login Issues

- **"Account is inactive"**: Kontak ang imong administratorâ€”ang imong account mahimong kinahanglan og activation

## Pag-Sabot sa Imong User Role

Human sa pag-login, ang sistema makaila sa imong **user role**. Ang imong role nag-determine:
- Unsa ang menu options nga makita nimo
- Unsa ang mga aksyon nga mahimo nimo
- Unsa ang mga vouchers nga makita nimo
- Unsa ang mga notifications nga madawat nimo

### Common Roles

- **REQUESTER**: Ikaw naghimo og disbursement vouchers
- **GSO**: Ikaw naghimo og procurement-related vouchers
- **HR**: Ikaw naghimo og HR-related vouchers
- **SECRETARY**: Ikaw nag-review ug nag-route sa vouchers
- **MAYOR**: Ikaw nag-approve sa vouchers
- **BAC**: Ikaw nag-review sa procurement vouchers
- **BUDGET**: Ikaw nag-verify sa budget allocation
- **ACCOUNTING**: Ikaw nag-validate sa financial information
- **TREASURY**: Ikaw nag-process sa payments
- **ADMIN**: Ikaw nag-manage sa users ug system settings

:::tip Role Information
Ang imong role gipakita sa imong profile o dashboard. Kung dili ka sigurado bahin sa imong role o permissions, kontak ang imong system administrator.
:::

## Dashboard Overview

Human sa pag-login, makita nimo ang imong **Dashboard**â€”ang imong home page sa sistema. Ang dashboard nagpakita:

### Key Sections

1. **Welcome Message**: Personalized greeting uban sa imong ngalan
2. **Quick Statistics**: Overview sa vouchers nga relevant kanimo
3. **Pending Actions**: Items nga nanginahanglan sa imong atensyon
4. **Recent Activity**: Latest updates sa vouchers nga involved ka
5. **Status Summary**: Count sa vouchers by status

### Unsa ang Makita Nimo (Role-Specific)

#### Para sa Requesters, GSO, ug HR Users
- Number sa vouchers nga imong gihimo
- Status sa imong vouchers (Draft, Pending, Approved, ug uban pa)
- Recent voucher activity

#### Para sa Reviewers (Secretary, Mayor, BAC, Budget, Accounting)
- Number sa vouchers pending sa imong review
- Vouchers nga naghulat sa imong aksyon
- Recently reviewed vouchers

#### Para sa Treasury Users
- Vouchers ready para sa check issuance
- Vouchers available para sa release
- Recently released vouchers

#### Para sa Admin Users
- System-wide statistics
- User management overview
- Directory management status

## Navigation Menu Guide

Ang **navigation menu** kasagaran nahimutang sa top sa page o sa sidebar. Kini naglangkob sa links sa lain-laing sections sa sistema.

### Common Menu Items

- **Dashboard**: Balik sa imong home page
- **Disbursements**: Tan-awa ug manage ang vouchers
- **Create Request**: Himoa ang bag-ong voucher (para sa Requesters, GSO, HR)
- **My Vouchers**: Tan-awa ang vouchers nga imong gihimo (para sa Requesters, GSO, HR)
- **Pending Reviews**: Vouchers nga nanginahanglan sa imong review (para sa Reviewers)
- **User Management**: Manage ang users (Admin lang)
- **Directory Management**: Manage ang payees, tags, items (Admin lang)
- **System Settings**: Configure ang system settings (Admin lang)

:::info Menu Visibility
Ang menu items nga makita nimo nagdepende sa imong role. Ayaw kabalaka kung dili nimo makita ang ubang optionsâ€”normal na base sa imong permissions.
:::

## Profile Management Basics

### Pag-Access sa Imong Profile

Tan-awa ang imong ngalan o profile icon sa top-right corner sa page. I-click kini aron ma-access:
- Ang imong profile information
- Account settings
- Logout option

### Important Profile Information

- **Imong Ngalan**: Gipakita sa tibuok sistema
- **Imong Role**: Nag-determine sa imong permissions
- **Imong Department**: Importante para sa Requesters (nag-determine kung unsa ang vouchers nga mahimo nimo himoon)
- **Email**: Gigamit para sa notifications (kung configured)

### Pag-Logout

Kung tapos na ka sa paggamit sa sistema:
1. I-click ang imong profile/ngalan sa top-right
2. I-click ang **Logout** o **Sign Out**
3. Ma-return ka sa login page

:::warning Security Best Practice
Kanunay mag-logout kung tapos na ka, labi na sa shared computers. Kini nag-protektar sa imong account ug sa sistema.
:::
